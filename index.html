Before you start
  • Chrome or Edge browser (required for Web Serial)
  • USB data cable — not a charge-only cable
  • Close Arduino IDE or any other app using the serial port
  • Hold the BOOT button on your CYD while clicking Connect, then release after selecting the port

Troubleshooting
No serial port appears / connection fails immediately
Make sure you're holding BOOT on the CYD when you click Connect and while selecting the port — release it only after the port dialog closes. Use a data USB cable, not a charge-only cable. Try a different USB port. On Windows, install the CH340 driver if the port doesn't appear at all. Mac users typically don't need drivers.
Install fails immediately or gets stuck
Hold the BOOT button while clicking Install. On some boards: hold BOOT, tap RST briefly, then release BOOT. Close Arduino IDE or any Serial Monitor before trying again.
Screen is blank after installing
Unplug and replug the cable. If still blank, hold BOOT + RST together for 3 seconds, then release. The device should restart normally.
