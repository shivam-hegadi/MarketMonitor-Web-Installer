Before you start
  • Chrome or Edge browser (required for Web Serial)
  • USB data cable — not a charge-only cable
  • Close Arduino IDE or any other app using the serial port
  • Hold the BOOT button on your CYD while clicking Install, release after selecting the port
1
Install firmware
Hold BOOT on the CYD, click Install, select USB Serial from the browser dialog, then release BOOT. The installer will handle everything — don't unplug during the process.
2
Set up your device
Firmware installed. Unplug and replug your device, then follow the guide to connect to WiFi and add stocks.

Troubleshooting
No serial port appears in the dialog
Hold BOOT on the CYD before clicking Install and keep holding until after you select the port. Use a data USB cable, not charge-only. Try a different USB port. On Windows, install the CH340 driver. Mac users typically don't need drivers.
Install fails or gets stuck
Hold BOOT, tap RST briefly, then release BOOT — this forces the ESP32 into download mode. Close Arduino IDE or any Serial Monitor before retrying.
Screen is blank after installing
Unplug and replug the cable. If still blank, hold BOOT + RST for 3 seconds, then release.
